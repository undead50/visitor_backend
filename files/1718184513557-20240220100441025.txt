Core Banking System: Core banking systems are back-end systems that process daily banking transactions and post updates to accounts and other financial records. Core banking systems typically include deposit, loan, and credit processing capabilities, with interfaces to general ledger systems and reporting tools. Software is added-on to the core to support bank functionality that relies on the underlying account information 4.

BPM Solution (Nintex K2, SharePoint): BPM (Business Process Management) solutions are used to automate and manage business processes. Nintex K2 is a workflow platform that helps businesses automate their processes, while SharePoint is a web-based collaborative platform that integrates with Microsoft Office and provides document management, document collaboration, and business intelligence capabilities.
MIS/BI (SSRS): MIS/BI (Management Information System/Business Intelligence) is a system that collects, stores, and analyzes business data to support decision-making and strategic planning. SSRS (SQL Server Reporting Services) is a tool for creating and managing reports in Microsoft SQL Server.
ETL (SSIS): ETL (Extract, Transform, Load) is a process in data warehousing, ETL involves pulling data out of different sources, transforming it to fit operational needs (such as data type conversions, removing duplicates, etc.), and then loading it into a database. SSIS (SQL Server Integration Services) is a platform for data integration and workflow applications.
RPA Management System: RPA (Robotic Process Automation) is a technology that uses software robots or "bots" to automate routine tasks. An RPA Management System is a tool that manages, monitors, and optimizes these automation tasks.
Microsoft Azure Machine Learning: Microsoft Azure Machine Learning is a cloud-based service that allows developers and data scientists to build, train, and deploy machine learning models. It supports automated machine learning and integrates with existing DevOps processes. It also supports responsible machine learning, allowing users to understand models with interpretability and fairness, and protect data with differential privacy and confidential computing 5.
